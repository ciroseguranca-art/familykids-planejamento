FamilyKids • Planejamento Financeiro 2026

Aplicativo inspirado na sua planilha: despesas, fornecedores, fluxo de caixa, CDB, metas e saldo mensal.

Resumo mensal

Força das vendas x meta

A barra compara vendas acumuladas com a meta alvo calculada.

Leitura automática

Despesas e fornecedores

Valores negativos representam pagamentos/saídas. O app soma fornecedores, despesas e total de cada mês automaticamente.

Fluxo de caixa

Informe vendas, Pronampe e rendimento/aporte CDB. O total de saída vem da tela de despesas.

Configurações e metas

Meta mínima = fornecedores + despesas. Meta alvo = fornecedores × multiplicador + despesas.

Backup e uso

Este aplicativo salva os dados no próprio navegador deste aparelho. Para trocar de celular/computador, exporte o backup.

Para hospedar: envie este arquivo index.html para Netlify, Vercel ou GitHub Pages. Também funciona offline, abrindo o arquivo no navegador.
