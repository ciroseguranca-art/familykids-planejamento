APLICATIVO FAMILYKIDS - PLANEJAMENTO 2026

Como usar:
1. Abra o arquivo index.html no navegador do celular ou computador.
2. Edite vendas, despesas, fornecedores, CDB e metas.
3. Os dados são salvos no próprio navegador.
4. Use Exportar backup para guardar os dados e Importar backup para restaurar em outro aparelho.

Como hospedar:
- Netlify: arraste a pasta do aplicativo para o Netlify Drop.
- GitHub Pages/Vercel: publique esta pasta como site estático.

Observação:
Esta é uma versão inicial de aplicativo web baseado na estrutura da planilha enviada.
